# How to project with `Motia`

## Go to root directory
```sh
cd code
mkdir motia && cd motia
```

## Create project
```sh
npx motia@latest create
```

```sh
? What template do you want to use? (Use arrow keys) Starter (All languages; TS/JS + Python)
? Project name (leave blank to use current folder): motia-first
```
