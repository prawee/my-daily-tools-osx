# How to installation CLI of Bun

## Checking

```bash
bunx
# zsh: command not found: bunx
```

```bash
bun --version
# 0.1.2
```

## Installing

```bash
curl -fsSL https://bun.sh/install | bash
```

## Re-Checking

```bash
bunx
usage: bunx [--bun] package[@version] [...flags or arguments to pass through]
```
