# How to installation Sequepro

## Download

- Go to <https://sequelpro.com/download>
- Press `Download` button

## Installation

- Open your terminal
```bash
open ~/Downloads/sequel-pro-1.1.2.dmg
```
- Drag `Sequel Pro` to `/Applications/` folder

## Usage

- Open your terminal
```bash
open /Applications/Sequel\ Pro.app
```
