# How to update CLI

## Current
```bash
rustc --version
# rustc 1.78.0 (9b00956e5 2024-04-29)
cargo --version
# cargo 1.78.0 (54d8815d0 2024-03-26)
```

## Upgrade
```bash
rustup update
```
