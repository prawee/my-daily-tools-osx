# How to install CLI of `Vercel`

## Require
- Node version >= 18.0.0 and <=20.x.x

## Checking
```bash
vercel
# zsh: command not found: vercel
```

## Install
```bash
yarn global add vercel@latest
# success Installed "vercel@39.3.0" with binaries:
#       - vc
#       - vercel
# âœ¨  Done in 13.17s.
```

## Checking again
```bash
vc --version | vc -v | vercel --version | vercel -v
# Vercel CLI 39.3.0
# 39.3.0
```
