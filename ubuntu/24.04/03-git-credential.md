# How to set `Credential` of Git

## Configure
```sh
git config --global credential.helper store
```

## Fist time
Pull or Push

```sh
git pull
```
```sh
Username for 'https://github.com': prawee
Password for 'https://prawee@github.com': 
Already up to date.
```

## Next time
```sh
git pull
```
```sh
Already up to date.
```
