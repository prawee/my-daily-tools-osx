# Create project with Nextjs

## Create

```bash
kool create nextjs kool-nextjs
```

## Start

```bash
cd kool-nextjs
kool start
# Creating network "kool-nextjs_kool_local" with the default driver
# Creating kool-nextjs_app_1 ... done
```

## Status

```bash
kool status

| SERVICE | RUNNING | PORTS                                     | STATE         |
```
