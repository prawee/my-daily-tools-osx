# Python 3.x

## Check version

```bash
python --version
# Python 2.7.10
```

## Install with brew

```bash
brew update
brew upgrade
brew install python3
```

## Recheck version again

```bash
python --version
# Python 2.7.10
python3 --version
# Python 3.6.2
```
