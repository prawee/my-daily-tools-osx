# How to install Nginx

## Check package

```bash
brew search nginx
```

## Install via brew

```bash
brew install nginx
```

## Start

```bash
nginx  | brew services start nginx
```

## Note

```bash
# Document root => /usr/local/var/www
# Default port => /usr/local/etc/nginx/nginx.conf
# Load config => /usr/local/etc/nginx/servers/
# Log   => /usr/local/var/log/nginx
```
