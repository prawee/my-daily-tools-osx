# Install Homebrew

## XCode

```bash
xcode-select --install
# press OK
```

## Homebrew

```bash
ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
# press enter | return
```

## Check conflicts and update package

```bash
brew doctor
brew update
brew upgrade
```
