# How to install imagesnap for opencv

## Checking first

```bash
imagesnap
# zsh: command not found: imagesnap
```

## Installation

```bash
brew install imagesnap

# ==> Downloading https://homebrew.bintray.com/bottles/imagesnap-0.2.6.mojave.bottle.1.tar.gz

brew list

```
