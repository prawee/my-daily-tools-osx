# How to install Hugo command with Brew

## Check first

```bash
hugo
#zsh: command not found: hugo
```

## Installation

```bash
brew list
brew install hugo
hugo version
# Hugo Static Site Generator v0.30.2 darwin/amd64 BuildDate: 2017-11-12T15:18:47+07:00
```
