# How to installation Dropbox with Brew

## Search

```bash
brew search dropbox
# ==> Formulae
# dropbox-uploader
# ==> Casks
# dropbox                                                                          homebrew/cask-versions/dropbox-beta
```

## Installation

```bash
brew cask install dropbox
```

## Usage

```bash
open /Applications/Dropbox.app/
```

Then go to `System Preferences` and `Security & Privacy`
