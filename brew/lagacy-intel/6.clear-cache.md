# How to clear cache of Brew

## Remove once time

```bash
brew cleanup
```

## Help

```bash
brew cleanup -h
```

```bash
#--prune
#-n, --dry-run
#-s
#--prune-prefix
#-v, --verbose
#-d, --debug
#-h, --help
```

## Save your storage

```bash
brew cleanup --prune=1
```
