# Install memcached

## Installation via brew

```bash
cd
brew update
brew upgrade
brew install memcached
```

## Start services

```bash
brew services list
brew services start memcached
```

## Uninstall memcached

```bash
brew uninstall --ignore-dependencies memcached
# Uninstalling /usr/local/Cellar/memcached/1.5.12... (11 files, 198.9KB)
```
