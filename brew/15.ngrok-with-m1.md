# How to installation NGROK with M1

## Checking

```bash
ngrok
# zsh: command not found: ngrok
```

## Installation

```bash
brew search ngrok
# ==> Formulae
# grok

# ==> Casks
# ngrok
```

```bash
arch -arm64 brew install ngrok
# download
# install
```

## Version

```bash
ngrok -v
#ngrok version 2.3.40
```
