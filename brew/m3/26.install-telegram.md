# How to install `Telegram` with M3

## Checking
```sh
telegram
```
```sh
zsh: command not found: telegram
```

## Searching
```sh
brew search telegram
```
```sh
==> Formulae
telegram-downloader                      telegraf

==> Casks
forkgram-telegram (deprecated)           telegram-desktop
telegram                                 telegram-desktop@beta
telegram-a
```

## Installing
```sh
brew install --cask telegram-desktop
```

...
