# How to install `Typst` 

## Checking
```bash
typst
# command not found: typst
```

## Install by Brew
```bash
arch -arm64 brew install typst
```

## Checking again
```bash
typst
# Welcome to Typst, we are glad to have you here! ❤️
```

## Version
```bash
typst -V | typst --version
# typst 0.12.0
```
