# How to create `Bot` with Telegram

## Open your telegram app

## Search input
  1. go to `Search` input
  2. typing `botfather`
  3. click `Start` button
     
## Create Bot
  1. write a message `/newbot`
  2. Choose a name `น้องกุ้งเฟี้ยว`
  3. Choose a username (end in `bot`) `openclaw_brt_bot`
  4. Copy `HTTP API` token
