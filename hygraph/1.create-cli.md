# Create GraphCMS CLI

## Checking

```bash
graphcms
# zsh: command not found: graphcms
```

## Install package

```bash
npm install -g @graphcms/cli
```

## Usage

```bash
graphcms
```

```
GraphCMS CLI

VERSION
  @graphcms/cli/0.0.2 darwin-arm64 node-v16.19.0

USAGE
  $ graphcms [COMMAND]

COMMANDS
  create  Generate GraphCMS project.
  help    Display help for graphcms.
  list    List available templates.
```
