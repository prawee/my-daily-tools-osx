# How to initialize `Typescript` and configure

## Install package and dependencies
```sh
npm init -y
npm install -D typescript @types/node mastra@latest
npm install @mastra/core@latest zod@^4
```

## Ignore files
```sh
nano .gitignore
```
```sh
node_modules
# Ctrl + O > Enter
# Ctrl + X
```
```sh
git add .gitignore
git commit -m "create gitignore file"
```

## Configure
```sh
nano package.json
```
```sh
{
  "scripts": {
    ...
    "dev": "mastra dev",
    "build": "mastra build"
  }
}
```
```sh
git add .
git commit -m "initialize package and configure scripts"
```

## Typescript
```sh
nano tsconfig.json
```
```sh
{
  "compilerOptions": {
    "target": "ES2022",
    "module": "ES2022",
    "moduleResolution": "bundler",
    "esModuleInterop": true,
    "forceConsistentCasingInFileNames": true,
    "strict": true,
    "skipLibCheck": true,
    "noEmit": true,
    "outDir": "dist"
  },
  "include": ["src/**/*"]
}
```
